# Launch Strategy

tokmax should launch as a product-led artifact.

Core message:

> Run one command and see what your Codex + Claude Code usage would have cost at API prices.

Channels:

- X.
- GitHub.
- Hacker News.
- Product Hunt.
- Relevant developer communities.

Trust messages:

- Only aggregate token counts, dates, machine labels, and computed cost are sent.
- Prompts, code, file contents, tool output, raw logs, and keys never leave the machine.
- The source is public and inspectable.

Readiness gates:

- Production Convex deployment is live.
- Publish endpoint abuse controls are active.
- README and web copy match the actual payload.
- A clean-machine end-to-end publish has been verified.
